# 🚀 Fivnet - Website Institucional Reformulado

## 📌 Sobre o Projeto

Este projeto consiste na reformulação completa do site institucional da **Fivnet**, uma empresa especializada em soluções de infraestrutura de tecnologia, redes e energia.

A proposta foi modernizar totalmente a presença digital da empresa, mantendo sua identidade visual (logo e cores), porém elevando o nível de design, experiência do usuário e percepção de mercado.

O resultado é um site com aparência **premium, tecnológica e corporativa**, alinhado com padrões atuais de empresas do setor.

---

## 🎯 Objetivos da Reformulação

* Modernizar o layout e a experiência do usuário
* Eliminar aparência de site genérico
* Criar uma identidade visual mais forte e tecnológica
* Melhorar a navegação e organização das informações
* Aumentar a percepção de credibilidade e profissionalismo
* Implementar efeitos visuais modernos (parallax, animações, transições)

---

## 🧱 Tecnologias Utilizadas

* HTML5 (estrutura semântica)
* CSS3 (estilização moderna e responsiva)
* JavaScript (interações e comportamento)
* PHP (envio de formulário de contato)
* Template base: HTML5 UP - Miniport (fortemente customizado)

## 🚀 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/fivnet-site.git
```

2. Abra o projeto:

```bash
cd fivnet-site
```

3. Execute em ambiente local:

* Abra o `index.html` no navegador
  ou
* Utilize um servidor local (XAMPP, WAMP, Live Server)

4. Para funcionamento do formulário:

* Configure um servidor com suporte a PHP
